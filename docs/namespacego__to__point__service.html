This the go_to_point_service node!

Author
Farouk Gasmi farou.nosp@m.k.li.nosp@m.onel1.nosp@m.00@g.nosp@m.mail..nosp@m.com
Version
1.0
Date
25/05/2023

Description: This code is a ROS (Robot Operating System) node that implements a simple go-to-point behavior for a robot. It uses laser scan data and odometry information to navigate the robot to a desired position.
